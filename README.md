# tools_portal

## Overview
A centralized location to increase development efficiency.

## Install
```bash
# Clone the project from:
https://github.com/tonylixu/tools_portal.git
 
# Install npm modules
$ cd Tools_Portal
$ npm install
 
# Install bower packages
$ cd app
$ bower install
 
# Run gulp default
$ gulp default
 
# Star the service
$ node server/server.js -p -t
```
